Este dataset contine imagenes preprocesadar utilizadas para entrenar y evaluar el modelo de reconocimiento de digitos escritos a mano, 
ES EL CONJUTOS DE DATOS QUE SE USO. Las imagenes han sido normalizadas y redimensionadas a un tamaño uniforme para facilitar 
el entrenamiento del modelo. Cada imagen corresponde a un digito del 0 al 9, y se han etiquetado adecuadamente para su uso en 
tareas de clasificación. Este dataset es ideal para experimentar con algoritmos de aprendizaje automático y redes neuronales 
en el campo del reconocimiento de patrones.
